# Flask Shop (Навчальний проєкт)

Простий навчальний інтернет-магазин на Flask з кошиком, який працює через `session`.

Проєкт створений **виключно для навчання** та демонструє:
- базову структуру Flask-застосунку
- роботу з маршрутами
- використання шаблонів Jinja2
- роботу з `session`
- додавання та видалення товарів з кошика

---

##  Функціонал

- Головна сторінка зі списком товарів
- Додавання товарів до кошика
- Перегляд кошика
- Видалення товарів з кошика
- Очищення кошика
- Підрахунок загальної вартості

---

##  Використані технології

- Python 3
- Flask
- HTML / Jinja2
- Session (Flask)
